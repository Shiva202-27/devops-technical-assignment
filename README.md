# DevOps Technical Assignment

## Tech Stack
- AWS EC2 (Ubuntu)
- Docker
- Node.js
- Nginx
- GitHub Actions
- Amazon CloudWatch
- Amazon S3
- k6

## Deployment
1. Launch EC2.
2. Install Docker and Node.js.
3. Build and run Docker container.
4. Configure Nginx reverse proxy.
5. Configure CloudWatch Agent.
6. Push code to GitHub to trigger GitHub Actions.

## Monitoring
CloudWatch dashboard with CPU, Memory, Disk and Network metrics.

## Load Testing
k6 used to validate application performance.
